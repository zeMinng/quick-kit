# 🧰 QuickKit | 在线实用快速工具箱

**QuickKit** 是一款追求极致体验的高性能纯前端实用工具集合。它打破了传统工具对后端的依赖，让“文件不出本地”成为现实，简洁高效，即用即走。

## 🌟 核心能力

*   🖼️ **图像处理**：格式互转、尺寸调整、智能压缩、网格拼接
*   🔄 **格式转换**：图片、文档、数据格式一键转换
*   📝 **JSON 处理**：格式化、校验、压缩、解析、高亮展示
*   ⚡ **实时预览**：所见即所得，调整参数实时渲染处理效果

## 🚀 为什么选择 QuickKit？

| 维度 | 传统在线工具 | QuickKit (本项目) |
| --- | --- | --- |
| **安全性** | 文件需上传至服务器，有隐私风险 | **100% 本地处理**，数据不出浏览器 |
| **速度** | 受限于网速和服务器带宽 | **极速响应**，处理速度仅取决于本地算力 |
| **费用** | 常见会员制、次数限制或水印 | **永久免费**，无任何套路 |
| **离线使用** | 必须联网 | 支持 PWA，**断网也能用** |

## 🛠️ 技术实现

本项目采用前沿的前端技术栈，确保极致处理性能：

* **核心处理**：Canvas API、Web Workers 实现硬件加速与无阻塞界面
* **格式支持**：集成高效编解码库，支持图片、JSON 等多种格式
* **高性能架构**：纯前端运行，不依赖任何后端服务
* **未来规划**：**将引入 Rust + WebAssembly** 重构 JSON 解析、格式转换等核心模块，进一步提升解析速度与处理效率，实现原生级性能体验

## 🏃 快速上手

### 1. 环境准备

确保你本地安装了 Node.js 环境，并使用 pnpm 作为包管理器。

### 2. 启动项目

```bash
# 克隆仓库
git clone 你的仓库地址

# 安装依赖
pnpm install

# 开启开发服务器
pnpm dev
```

## 🧩 Monorepo 结构

- `apps/web`: Web 应用（React + Vite）
- `packages/ui`: 共享 UI 组件与样式
- `engines`: Rust 引擎（预留 Wasm 方向）
